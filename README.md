# UZ API Generator

O‘zbek tilida ishlaydigan oddiy **shaxsiy API kod generatori**.  
Bir necha endpoint qo‘shib, Node.js + Express uchun tayyor kodni bir soniyada oling.

### Xususiyatlari
- To‘liq bitta HTML fayl (hech qanday server kerak emas)
- O‘zbek tilida interfeys
- GET, POST, PUT, DELETE methodlarini qo‘llab-quvvatlaydi
- JSON javob misolini qo‘lda kiritish mumkin
- Kodni nusxa olish tugmasi

### Qanday ishlatish
1. `index.html` faylini yuklab oling
2. Brauzerda oching
3. API nomini va endpointlarni kiriting
4. "API Kodini Generatsiya Qilish" tugmasini bosing
5. Tayyor kodni nusxa olib, o‘z loyihangizda ishlating

### Texnologiyalar
- HTML + Tailwind CSS + Vanilla JavaScript

### Kelajakdagi rejalari
- Python (FastAPI) qo‘shish
- Go tilini qo‘shish
- OpenAPI/Swagger JSON chiqarish
- Dark mode
